## Overview

This project builds upon the code developed in **Lab 2** and extends it to create **LABrador**, an interactive Mini Pupper companion robot. Additional functionality includes voice recognition, movement control, facial expressions, and audio feedback.

## Notes

The primary file that was modified for this project and should be graded is:

* **`sample_controller.py`**

This file contains the implementation for:

* Voice command recognition
* Command processing
* Robot movement behaviors
* Facial display updates
* Audio (bark) integration

## Comments

* The project is based on the Lab 2 starter code and existing Mini Pupper framework.
* Supporting image and sound assets are located in the corresponding **`my_images`** and **`sounds`** folders.
* No additional setup beyond the provided project dependencies is required.
